# Aszist

Open-source helpdesk support system

## Getting Started

* git clone git://github.com/philtr/aszist.git
* cd aszist
* bundle install
* rake db:migrate
* rails s

## Requirements

Rails 3.1.0

### Application Gems
* devise
* cancan
* hashie
* mailman
* mysql2
* paper_trail
* redcarpet
* simple_form
* sqlite3

### Asset template engines
* sass-rails
* coffee-script
* uglifier
* jquery-rails
* therubyracer
* 'mail_view
